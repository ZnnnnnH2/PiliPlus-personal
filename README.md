# PiliPlus (精简修改版)

此版本为 PiliPlus 的精简修改版，在原项目基础上进行了修改，移除了动态、首页、搜索页面的大部分内容，旨在提供一个更轻量的体验，不要再沉浸于短视频。

本项目主要用于个人学习和特定功能测试。
